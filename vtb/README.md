# ВТБ

### Описание
Личный кабинет агентов и партнеров

### Основной стек (frontend)

Построен на микрофронтендах с использованием module federation от webpack

- react
- typescript
- mobx
- собственный ui-kit
- xlsx
- d3js
- husky
- lint-staged
